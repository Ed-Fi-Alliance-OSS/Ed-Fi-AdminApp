# Development Toolkit

## Applications to Install

- Node.js 22
- Docker Desktop (Windows / MacOS) or equivalent

## Frameworks

- [NestJs](https://docs.nestjs.com/) for the API application.
  - [node-config](https://github.com/node-config/node-config) manages configuration files.
- [React](https://react.dev) for the front-end application.
  - [Chakra UI](https://chakra-ui.com/) is a modular and accessible component library for React applications. Using [version 2](https://v2.chakra-ui.com/).
  - [Storybook](https://storybook.js.org/) is a tool for developing and documenting UI components in isolation, enabling visual testing and design review.
  - [Vite](https://vitejs.dev/) is a fast build tool and development server for modern web projects, providing instant hot module replacement and optimized production builds.

## Tooling

- [Jest](https://jestjs.io/) for JavaScript/TypeScript testing.
- [Nx](https://nx.dev/) for monorepo management and task running.
- [TypeORM](https://typeorm.io/) for object-relational mapping and database management in the API.
